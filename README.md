# Game Of Greed

**Author**: Tammy Do
**Version**: 1.0.0

## Overview
The game Game Of Greed is a popular family game. It is also known as Ten Thousand, Zilch or Foo. The programming of this game is a great demonstration of core Python tools and the basics of the language.

The features of this program includes 
*Rolling between 1 and 6 dice
*User ability to set aside dice each roll
*User ability to enter their score per roll
*User ability to “bank” current score or roll again
*Tracking of total score
*Tracking of current round

For rules of gameplay, refer to the Wikipedia page: https://en.wikipedia.org/wiki/Dice_10000

## Getting Started
*A user must have python installed on the machine.
*Users can clone down the game-of-greed repository and run the game in a python editor or terminal
